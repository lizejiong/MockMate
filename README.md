# AI Interview Coach

AI Interview Coach 是一个面向真实上线的 Expo React Native 项目骨架。产品采用 BYOK 模式，帮助用户完成简历诊断、模拟面试、每轮点评、评估报告、学习模式和弱点追踪。

## 快速开始

```bash
pnpm install
pnpm mobile:start
```

Node 版本使用 20 LTS，建议通过 `.nvmrc` 对齐到 `20.19.0`。

## 常用校验

```bash
pnpm mobile:typecheck
pnpm mobile:lint
pnpm mobile:test
```
